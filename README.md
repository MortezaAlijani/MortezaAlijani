# Morteza Alijani

Ph.D. Researcher at imec-WAVES, Ghent University.

My research focuses on intelligent sensing, localization, estimation, signal processing, and autonomous systems, with applications in visible light positioning and sensing (VLP/S), IoT-enabled localization, and AI-driven sensing technologies.

## Research Interests

- Intelligent Sensing
- Localization and Estimation
- Signal Processing
- Autonomous Systems
- Machine Learning for Sensing and Localization
- Visible Light Positioning and Sensing (VLP/S)

## Research Topics

- AI-driven visible light localization and sensing 
- Optical wireless sensing
- Sensor fusion and estimation
- Signal processing algorithms
- Intelligent positioning systems
