# Hi, I'm Morteza Alijani 👋

I am a **Ph.D. Researcher at imec-WAVES, Ghent University**, working at the intersection of intelligent sensing, localization, estimation, signal processing, and autonomous systems.

My research focuses on developing data-driven and model-based methods for **visible light positioning and sensing (VLP/S)**, optical wireless sensing, IoT-enabled localization, sensor fusion, and AI-driven sensing technologies.

## 🔬 Research Interests

* Intelligent sensing
* Localization and positioning
* Estimation and sensor fusion
* Signal processing
* Autonomous systems
* Machine learning for sensing and localization
* Visible light positioning and sensing
* Optical wireless sensing

## 📡 Current Research Topics

* AI-driven visible light positioning and sensing
* Optical wireless sensing systems
* Sensor fusion and state estimation
* Signal processing algorithms for localization
* Intelligent positioning systems
* IoT-enabled sensing and localization
* Machine learning for signal-based sensing applications
